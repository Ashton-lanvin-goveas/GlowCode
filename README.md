# GlowCode
GlowCode is a full-stack application with:
Frontend: React
Backend: Node.js (Express)
The repository does not include node_modules or environment files.
All dependencies are installed locally using the commands below.

Python dependencies:
The Python virtual environment (venv) is not included in this repository.
It must be created locally to install and isolate required Python packages.

Create and activate the virtual environment, then install dependencies:
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
